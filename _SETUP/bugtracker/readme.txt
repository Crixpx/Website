*** Install BugTracker **

1) Head into /SQL and execute all of them.
2) Open /Bugtracker then edit "flyspray.conf.php" with notepad. Change the following lines to match your database credentials.

[database]
dbtype = "mysqli"
dbhost = "localhost"
dbname = "bugtracker"
dbuser = "root"
dbpass = "ascent"
dbprefix = "cms_"

3) The default administrator account credentials are: 

Username: admin
Password: RebornBugTracker

Please change them after logging in!
